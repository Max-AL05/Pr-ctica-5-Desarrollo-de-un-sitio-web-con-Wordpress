Descripción del Proyecto Tienda en línea 100 % funcional especializada en la venta de claves digitales de videojuegos para PC (Steam, Epic, Origin), PlayStation, Xbox y Nintendo Switch. Desarrollada con WordPress y WooCommerce como práctica integral del curso de Desarrollo de Sitios Web – BUAP FCC (Otoño 2025). Características principales: Productos descargables con entrega inmediata Métodos de pago locales (Mercado Pago, PayPal, transferencia) Diseño responsive con Full Site Editing (tema Twenty Twenty-Five) Optimización SEO, velocidad y seguridad Gestión completa de categorías, atributos variables y cupones
