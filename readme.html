Tienda Virtual de Videojuegos

Proyecto educativo desarrollado con WordPress y WooCommerce, siguiendo el enfoque formativo del Mega Curso de WordPress de Cero a Maestro. Se aplican competencias de instalación, personalización visual (Gutenberg y FSE), administración de contenido, configuración de e‑commerce, seguridad y optimización SEO.

WordPress 6.x WooCommerce 8.x PHP ≥ 7.4 MySQL ≥ 5.7 Gutenberg Full Site Editing

Descripción del Proyecto

La tienda en línea se construye sobre WordPress con el plugin WooCommerce para ofrecer una solución de comercio electrónico completa. El estudiante instala y configura el entorno local, diseña la interfaz con Gutenberg y FSE, y prepara la operativa de la tienda (productos, inventario, impuestos, pagos y envíos). Además, se integran prácticas de seguridad, mantenimiento y SEO 2025 (estructura semántica, velocidad, Core Web Vitals, optimización de imágenes e indexación).

  • Instalación y configuración del entorno local.
  • Personalización visual con Gutenberg y Full Site Editing (FSE).
  • Administración del contenido y arquitectura de información.
  • Configuración integral de WooCommerce (productos, variantes, impuestos, pagos, envíos).
  • Seguridad, respaldos y actualización segura del sitio.
  • Optimización SEO orientada a Core Web Vitals y rendimiento.

Tema Elegido

Tienda Virtual de Videojuegos

Enfocada en la venta de videojuegos digitales y físicos, consolas y accesorios relacionados. La identidad visual utiliza un diseño moderno orientado a la comunidad gamer, con imágenes de alta calidad y una experiencia de usuario optimizada.

  • Categorías: PC, PlayStation, Xbox, Nintendo.
  • Productos: juegos, consolas, controles, tarjetas de regalo, accesorios.
  • UX/UI: navegación clara, búsqueda por plataforma, filtros por género y precio.

Pasos para Ejecutar el Sitio en Local

  1. Instalar un entorno local
    Descarga e instala en tu equipo.
  2. Crear y configurar el sitio WordPress
    Crea un nuevo sitio (por ejemplo: tienda-videojuegos), define credenciales de administrador, completa la instalación y accede al panel (/wp-admin).
  3. Instalar y configurar WooCommerce
    En Plugins > Añadir nuevo, instala y activa WooCommerce. Ejecuta el asistente: moneda, impuestos, métodos de pago (ej. Stripe/PayPal), zonas y tarifas de envío.
  4. Diseño con Gutenberg y FSE
    Crea páginas principales (Inicio, Tienda, Categorías, Contacto), patrones personalizados, plantillas (cabecera, pie) y un menú de navegación alineado con el objetivo comercial.
  5. Alta de productos
    Define categorías (PC, PlayStation, Xbox, Nintendo). Añade productos con imagen destacada, galería, precio, inventario, atributos (plataforma, idioma) y variantes (ediciones).
  6. SEO y rendimiento
    Instala un plugin SEO (por ejemplo, Yoast/Rank Math), optimiza imágenes (formato moderno, lazy‑load), revisa Core Web Vitals, habilita caché y limpia recursos innecesarios.
  7. Seguridad y mantenimiento
    Configura copias de seguridad periódicas, control de usuarios y políticas de actualización segura para temas y plugins.
  8. Acceder al sitio
    Abre el navegador y visita la URL local, por ejemplo: http://localhost/tienda-videojuegos (o la que genere tu entorno).
